 # WebScraper

## Installation

### Clone Repositories

``` bash 
git clone https://github.com/abin-karukappallil/Advanced-web-scraper
```
### Install Dependencies

``` bash
pip install -r requirements.txt
```

## Usage

``` bash
python main.py
```
